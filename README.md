# Employee Management System

A simple console-based Employee Management System built using Python and MySQL.

## Features
- Add Employee
- Display Employees
- Promote Employee
- Remove Employee

## Technologies Used
- Python
- MySQL
- mysql-connector-python

## How to Run
1. Install Python
2. Install MySQL Server
3. Create a database and employees table
4. Run the script:
   python employee_management.py
